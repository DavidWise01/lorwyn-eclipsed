LORWYN ECLIPSED (ECL) — an ARENA Magic: The Gathering sphere: Magic's 2026 RETURN to Lorwyn, the fae plane under a long eclipse, curated by basic worth. Pulls all ~273 cards LIVE from Scryfall (card data + the non-foil USD it mirrors from TCGplayer) and buckets them into worth tiers — 🩸 chase ($5+) · staples ($1–5) · role-players ($0.25–1) · bulk (&lt;$0.25) — with a set-value summary (sum / top card / chase count), search, rarity filter, foil-worth toggle, and a Scryfall thumbnail per card. The curation is the tiering + the read applied to REAL numbers (verified live: 273 cards, ~$246 non-foil set value, top Chronicle of Victory ~$18.91, 13 chase — with reprints like Bloom Tender &amp; Steam Vents carrying value).
